# Site — Empresa de Desenvolvimento de Softwares

Site institucional de uma empresa fictícia de desenvolvimento de software, com seções de serviços, diferenciais e formulário de contato.

## Tecnologias

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

## Seções da página

- Header com logo da empresa
- Banner com lista de serviços e formulário de contato
- Seção de diferenciais com ícones (Desktop, Web e Mobile)
- Rodapé

## Estrutura

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git

```
